Skip

明确不接入的场景

  • 分析页四段 tab（量化趋势 / 五维 / 关联 / 复盘）— 纵向 scroll + anchor，四段同页共存，不是同槽互斥视图。
  • 分析周期 preset（本周 / 本月 / 自定义）— Header 下划线文字，改 pill 滑块等于改 IA。
  • 日历月/周/日正文 — 三 shell 按 URL 只 mount 一个；pager 需三视图同时挂载 + 预取，成本高。
  • 访谈工作区切换（访谈 ↔ 完整日志 ↔ 当天评分）— 已有过渡卡 + autosave 边界；与五维切换不同，仍不接入 pager。
  • 8 项快扫网格 — 8 卡就地更新 sparkline，非两页互斥，ROI 低。
