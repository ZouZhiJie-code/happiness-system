Next Actions

下一步行动

产品验收主线已经收口，下一阶段应转向真实部署上线准备，尤其是中国大陆可访问场景下的合规、基础设施和生产环境验证。

冻结当前上线候选版本

以当前 main 的 merge commit 作为 release candidate，补一份发布说明：包含已上线范围、已知限制、环境变量清单和回滚方式。

确认中国大陆部署路径

确定域名、服务器或云平台、数据库、对象存储、HTTPS 证书和 CDN。若使用大陆服务器或 CDN，需要推进 ICP 备案；若使用境外服务，需要单独评估访问速度和稳定性。

配置生产环境变量和密钥

补齐数据库连接、Cookie/session 密钥、AI provider endpoint、日志级别、转写能力开关等。当前画像 fallback 已能处理 AI endpoint 缺失，但正式上线仍应配置真实 AI endpoint 并做效果验收。

执行生产部署前 smoke test

在预生产环境重复最小主链：注册登录、五维任一访谈、生成单篇日志、保存、生成当天完整日志、Calendar 回看、Analysis 查看、Profile 生成画像。

补齐运维监控与备份

至少准备错误日志、请求日志、数据库备份、部署回滚脚本和关键 API 健康检查。上线后第一批用户应以小流量灰度进入。

真实 AI 访谈效果继续抽样

虽然当前 AI 验收已通过，但真实用户语料会更发散。上线后要继续按五维抽样，重点看 gratitude / improvement / fulfillment 的抽取、收束和正文自然度。
